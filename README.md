# diagnosys
diagnosys is designed to be ran from ~/Documents

you can run with "bash ~/Documents/diagnosys-main/run.sh"

install the dependencies first, if not installed already

the dependencies are psutil 5.9.8, pygments 2.17.2, mdurl 0.1.2, markdown-it-py 3.0.0, and rich 13.7.0

you can install by running install_dependencies.sh or with "pip install psutil==5.9.8 pygments==2.17.2 mdurl==0.1.2 markdown-it-py==3.0.0 rich==13.7.0"
